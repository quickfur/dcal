dcal
====

A simple calendar program in D, to demonstrate its component-programming
features. Used as a case study in:

	http://wiki.dlang.org/Component_programming_with_ranges

Usage:

````
        dcal
        dcal 1971
        dcal jan
        dcal january
        dcal jan 1971
        dcal 1971 jan
        dcal 1971 1
````

Month names are case-insensitive.
